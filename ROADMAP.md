# ROADMAP

prompt-ide は、ファイル閲覧・プロンプト管理・Claude API 実行を 1 つにまとめた
Electron + React + TypeScript 製のデスクトップアプリです。

このドキュメントは「実装済みの機能」と「今後実装した方がよい機能」を整理したものです。

## 凡例

- `[x]` … 実装済み
- `[ ]` … 未実装（今後の実装候補）
